게시판 API 목록

GET
/api/news
1.게시판 리스트
curl -X GET /api/news
Body Type Required Description
page string optional [기본값 = 0] page 작성해줍니다.
limit string optional [기본값 = 10] 몇개의 records를 가져오는지 결정합니다.
orederBy string optional [기본값 = asc] asc,desc (오름차순,내림차순)
POST
/api/news
2.게시판 생성
curl -X POST /api/auth/news
--header "Content-Type: application/json" \
--header 'Authorization: Bearer Token' \
--data-raw '{ "title": "Title", "content": "Conetnet" }'
Body Type Required Description
title string Required 게시판의 title 입니다. (최소2 글자 최대 100글자 입니다)
content string Required 게시판의 content 입니다. (최소2 글자 최대 1000글자 입니다)
GET
/api/news/:id
3.single 게시판
curl -X GET /api/news/:id
PUT
/api/news/:id
4.single 게시판 업데이트
curl -X PUT /api/auth/news/:id
--header "Content-Type: application/json" \
--header 'Authorization: Bearer Token' \
--data-raw '{ "title": "Title", "content": "Conetnet" }'
Body Type Required Description
title string Required 게시판의 title 입니다. (최소2 글자 최대 100글자 입니다)
content string Required 게시판의 content 입니다. (최소2 글자 최대 1000글자 입니다)
DELETE
/api/news/:id
5.single 게시판 삭제
curl -X DELETE /api/auth/news/:id
--header 'Authorization: Bearer Token' \
